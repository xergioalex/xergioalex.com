Usando MDX

#Personal

Este tema viene con la integracion @astrojs/mdx instalada y configurada en tu archivo de configuracion astro.config.mjs. Si prefieres no usar MDX, puedes desactivar el soporte eliminando la integracion de tu archivo de configuracion.

Por que MDX?

MDX es un sabor especial de Markdown que soporta sintaxis JavaScript y JSX embebida. Esto permite la capacidad de mezclar JavaScript y componentes de UI en tu contenido Markdown para cosas como graficos interactivos o alertas.

Si tienes contenido existente escrito en MDX, esta integracion con suerte hara que migrar a Astro sea muy sencillo.
