Este tema viene con la integracion [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) instalada y configurada en tu archivo de configuracion `astro.config.mjs`. Si prefieres no usar MDX, puedes desactivar el soporte eliminando la integracion de tu archivo de configuracion.

## Por que MDX?

MDX es un sabor especial de Markdown que soporta sintaxis JavaScript y JSX embebida. Esto permite la capacidad de [mezclar JavaScript y componentes de UI en tu contenido Markdown](https://docs.astro.build/en/guides/markdown-content/#mdx-features) para cosas como graficos interactivos o alertas.

Si tienes contenido existente escrito en MDX, esta integracion con suerte hara que migrar a Astro sea muy sencillo.

## Ejemplo

Asi es como importas y usas un componente de UI dentro de MDX.
Cuando abras esta pagina en el navegador, deberas ver el boton clickeable a continuacion.

import HeaderLink from '@/components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicked!')">
	Componente embebido en MDX
</HeaderLink>
