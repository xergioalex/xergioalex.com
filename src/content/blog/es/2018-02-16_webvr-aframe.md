Reanudamos actividades con gran asistencia y dos charlas: yo sobre WebVR con A-Frame, y Alejandro Suárez sobre JavaScript point-free con RamdaJS. Me enfoqué en llevar la realidad virtual a la web — haciendo que experiencias inmersivas 360° sean accesibles para todos.

La web es la plataforma de distribución masiva más importante. A-Frame, un framework open-source de Mozilla, te permite crear contenido 3D y VR con HTML — sin pasos de construcción. Es compatible con la mayoría de bibliotecas y frameworks JavaScript existentes: React, Angular, D3.js, Vue.js y más.

## Lo Que Cubrí
