# Task 1: Create RelatedPosts Component

## Context

We need a Svelte component to display related blog posts. It should show 3-5 posts that share tags with the current post.

## Goal

Create `src/components/blog/RelatedPosts.svelte` component.

## Instructions

1. **Create** the component file at `src/components/blog/RelatedPosts.svelte`

2. **Implement** the component with:
   - Props: `posts` (array of related posts)
   - Grid layout (1 col mobile, 2-3 cols desktop)
   - Use BlogCard component for each post
   - Section heading "Related Posts"

3. **Follow** existing patterns from StaticBlogSearch.svelte

## Example Implementation

```svelte
<script lang="ts">
  import type { CollectionEntry } from 'astro:content';
  import BlogCard from './BlogCard.svelte';

  export let posts: CollectionEntry<'blog'>[] = [];
</script>

{#if posts.length > 0}
  <section class="mt-16 border-t border-gray-200 dark:border-gray-700 pt-8">
    <h2 class="text-2xl font-bold mb-6 text-gray-900 dark:text-white">
      Related Posts
    </h2>
    <div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
      {#each posts as post}
        <BlogCard {post} />
      {/each}
    </div>
  </section>
{/if}
```
