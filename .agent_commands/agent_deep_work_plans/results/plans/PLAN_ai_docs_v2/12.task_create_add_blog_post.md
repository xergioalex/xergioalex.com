# Task 12: Create add-blog-post Skill

## Context

Need a skill to help AI agents create blog posts with correct frontmatter and Content Collections patterns.

## Goal

Create `.claude/skills/add-blog-post/SKILL.md` for creating blog posts.

## Instructions

1. **Create** folder: `.claude/skills/add-blog-post/`
2. **Create** `SKILL.md` following the template pattern
3. **Include** guidance for:
   - Frontmatter schema (from content.config.ts)
   - File naming (kebab-case)
   - Tags usage
   - MDX for interactive content
   - Image references

## Skill Content

**Tier:** 1 (Light)
**Intent:** create

**Key Sections:**
- Frontmatter schema:
  ```yaml
  ---
  title: 'Post Title'        # Required
  description: 'Description' # Required
  pubDate: 'Jan 31 2026'     # Required
  updatedDate: 'Feb 1 2026'  # Optional
  heroImage: '/images/...'   # Optional
  tags: ['tech', 'personal'] # Optional
  ---
  ```
- File location: `src/content/blog/`
- Naming: `my-post-slug.md` or `.mdx`
- Tags must exist in `src/content/tags/`

**Validation:**
- `npm run astro:check`
- `npm run build`
