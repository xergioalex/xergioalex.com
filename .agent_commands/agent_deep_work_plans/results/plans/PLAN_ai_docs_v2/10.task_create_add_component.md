# Task 10: Create add-component Skill

## Context

Need a skill to help AI agents create new Astro or Svelte components correctly.

## Goal

Create `.claude/skills/add-component/SKILL.md` for creating components.

## Instructions

1. **Create** folder: `.claude/skills/add-component/`
2. **Create** `SKILL.md` following the template pattern
3. **Include** guidance for:
   - Choosing Astro vs Svelte
   - Component file structure
   - Props interface definition
   - Tailwind styling patterns
   - Dark mode support
   - Hydration directives (for Svelte)

## Skill Content

**Tier:** 1 (Light)
**Intent:** create

**Key Sections:**
- When to use Astro (.astro):
  - Static content
  - Server-rendered
  - No client-side interactivity
- When to use Svelte (.svelte):
  - Interactive UI
  - Client-side state
  - Event handlers
- Where to place:
  - `src/components/` (root for standalone)
  - `src/components/blog/` (blog-related)
  - `src/components/home/` (homepage sections)
  - `src/components/layout/` (layout components)

**Validation:**
- `npm run biome:check`
- `npm run astro:check`
