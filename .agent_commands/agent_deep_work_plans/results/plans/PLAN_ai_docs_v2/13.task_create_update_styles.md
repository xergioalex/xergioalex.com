Create `.claude/skills/update-styles/SKILL.md` for styling updates.

## Instructions

1. **Create** folder: `.claude/skills/update-styles/`
2. **Create** `SKILL.md` following the template pattern
3. **Include** guidance for:
   - Tailwind utility classes
   - Dark mode (`dark:` variants)
   - Custom CSS in global.css
   - Component-scoped styles
   - Responsive design (`sm:`, `md:`, `lg:`)

## Skill Content

**Tier:** 1 (Light)
**Intent:** fix/update

**Key Sections:**
- Tailwind patterns:
  - Use utility classes first
  - Custom classes in `@layer components`
  - Theme colors: `bg-main`, `text-secondary`
- Dark mode:
  - Always add `dark:` variants for colors
  - Test in both modes
- Files:
  - `src/styles/global.css` - Global styles
  - `tailwind.config.mjs` - Configuration
- Common patterns:
  ```html
  <div class="bg-white dark:bg-gray-900 text-gray-900 dark:text-gray-100">
  ```

**Validation:**
- `npm run biome:check`
- Visual inspection in `npm run dev`
