# Task 7: Create src/pages/README.md

## Context

The `src/pages/` folder contains all routes using Astro's file-based routing system.

**Structure:**
- index.astro - English homepage
- about.astro - About page
- contact.astro - Contact page
- rss.xml.js - RSS feed
- es/index.astro - Spanish homepage
- blog/
  - index.astro - Blog listing
  - [...slug].astro - Individual post pages
  - page/[page].astro - Pagination
  - tag/[tag].astro - Tag pages
  - tag/[tag]/page/[page].astro - Tag pagination
- api/
  - posts.json.ts - Blog search API

## Goal

Document the routing structure and page patterns.
