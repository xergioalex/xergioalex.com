## Lifecycle

### 1. **Creation**

Agent creates a plan based on your prompt:

```
.agent_commands/agent_deep_work_plans/results/plans/PLAN_ui_expansion/
```

### 2. **Execution**

Agent works through tasks sequentially:

- Marks tasks as `[x]` when complete
- Updates completion logs
- Commits after each task

### 3. **Completion**

All tasks finished - plan is complete!

### 4. **Cleanup**

After work is merged, you can delete the plan folder:

```bash
rm -rf .agent_commands/agent_deep_work_plans/results/plans/PLAN_ui_expansion/
```
